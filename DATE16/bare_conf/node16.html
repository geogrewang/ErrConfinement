Conclusion

In this work, probabilistic error masking matrix is first introduced which addresses the error masking effects of logic circuits. Taking advantage of PeMM, an fast and approximate error prediction framework is illustrated which tracks the paths of error propagation and fine-grained error probabilities. Vulnerability of hardware resources can be easily predicted through the framework. Benchmark results with state-of-the-art high-level fault simulation indicates that the proposed framework achieves high accuracy of error prediction and significant speedup. Both data and control flow errors can be approximately predicted.

Future work includes detailed QoS-reliability exploration for approximate applications using the framework. Besides, the setup of proposed framework for generic processors will be automated so that error propagation on application specific architectures can be fast and accurately analyzed.

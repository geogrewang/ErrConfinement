Targeting large logic blocks

PeMM performs a linear transformation of error masking probability from logic inputs to outputs, which is not the case for large logic blocks with internal data dependencies. To address such issue, larger circuits are decomposited into logic sub-blocks with individual PeMMs. Figure 3 indicates PeMM decomposition based on data dependencies, where the large logic block alu_ex is splitted into 3 sub-blocks. Signals alu_in1 and alu_in2 connect first two sub-blocks while alu_out connects the last two sub-blocks. PeMMs for sub-blocks are characterized individually. The intra token pool is used to keep the temporary tokens, which represent data with error probabilities, for processing on next logic sub-blocks.
